📄 [AI 아동 그림 인지 분석 웹사이트 - 화면 설계서 초안]
1. 서비스 개요

이 웹사이트는 30~40대 부모, 특히 어머니들을 주요 사용자로 하여, 아이가 그린 사람 그림을 업로드하면 AI가 인지적 특성을 분석하여 지능 수준을 시각화된 결과로 제공하는 원페이지 웹 서비스입니다.

2. 사용자 시나리오

부모는 웹사이트에 접속하여 아이 정보(성별, 생년월일)를 입력합니다.

그림 안내 문구를 보고 아이가 그린 그림을 업로드합니다.

AI가 분석을 시작하고 잠시 후 결과를 보여줍니다.

분석 결과는 나무와 정규분포 곡선을 통해 시각화되며, PDF로 저장하거나 공유할 수 있습니다.

3. 화면 구성 흐름 (원페이지 스크롤)

Hero 영역 – 소개 문구 및 분위기

아이 정보 입력 폼

그림 안내 섹션 (예시 그림/텍스트 포함)

그림 업로드 영역

분석 중 화면 (로딩 애니메이션 포함)

분석 결과 표시 (나무 + 정규분포 그래프)

결과 저장/공유 버튼

4. 주요 화면별 구성 요소

아이 정보 입력: 이름, 성별, 생년월일 입력 폼

그림 안내: 그릴 그림 종류 안내 (텍스트 또는 예시 이미지)

업로드 박스: 그림 파일을 업로드하는 버튼

분석 중: 간단한 텍스트 + 로딩 애니메이션

결과 화면:

원본 그림 (정사각형)

지능 나무 시각화 (물이 차는 듯한 컵 비유)

정규분포 곡선 시각화 (퍼센타일 위치 표시)

해석 문장, 날짜, 아이 정보 포함

5. 시각화 구성 요소 설명

나무 높이: 아이의 전체 추정 지능 수준 (위로 올라갈수록 높음)

컵에 물이 차오르는 표현: 추정치 시각화 (비율 느낌)

정규분포 곡선: 퍼센타일 시각화

좌측 그림 영역: 아이가 그린 원본 그림 그대로 표시

6. 결과 저장 및 공유 기능

결과는 1회성 분석 결과로 서버에 저장되지 않음

PDF 다운로드 제공

링크 공유 가능성 고려 (비로그인 사용자도 가능하게)
